Getting Started

To get started with BeeMod, there isn't much you have to do. Just make sure the 'BeeMod' role is above the member role, this give the bot the required permissions to kick and ban the bot.

Economy & Levelling

BeeMod stores all information on your economy and levelling profile in json files. This is an easy database to use and is supported by discord.py and repl.it.

Commands

All BeeMod commands. Use b!help in Discord for in-depth help

Utilities

b!gstart - starts a giveaway
b!lvl - see your server level
b!servers - see how many servers the bot is in
b!website - shows the bot's website
b!news - view the latest BeeMod updates

Moderation

b!kick - kicks a member
b!ban - bans a member
b!unban - unbans a member
b!clear - clears a certain number of messages

Economy

b!bal - check your wallet and bank balance
b!beg - beg for some money
b!dep - deposit some money
b!withdraw - withdraw some money
b!slots - gamble for some money
b!shop - view the shop
b!buy - buy an item from the shop
b!bag - view the items you have bought from the shop
b!send - send someone some money
b!lb - view the global leaderboard

Fun

b!8ball - ask the 8ball a question
b!meme - sends a random meme from r/memes
b!rps - play rock paper scissors with beemod
b!inspire - send an inspirational quote
